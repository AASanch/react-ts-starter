# react-ts-starter
React With Typescript Starter Kit

Stack:

* Build:  Webpack
* CSS:  Sass
* Lint:  TSLint
* Transpiler:  Babel | Typescript
* EditorConfig
* UI: React
* Dev Server: Webpack Dev Server

Notes:
* I used VS Code so the project includes a .vscode folder with a settings.json and a launch.json file.
